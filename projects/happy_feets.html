Happy Feets – eCommerce Site to Sell Shoes

Loading...

Happy Feets is an e-commerce website selling footwear, featuring product views, payment integration, returns, and more.

Loading...

Business Impact:

Happy Feets is an e-commerce platform that brings seamless shopping experiences to users. The integration of Razorpay facilitates secure payments, while AWS hosting ensures reliable performance. With features like refund and return management, it provides both convenience and trust for customers.

Loading...

Additional Information:

The platform is hosted on AWS, with Route53 and Nginx for domain management and load balancing. Gunicorn ensures optimal performance for Django, and the use of PostgreSQL guarantees data integrity and fast processing.

Loading...

Features:

  • Fully functional e-commerce website.
  • Features like login, cart, wishlist, Razorpay payment, refund, return, and more.
  • Built with Python, Django, React, PostgreSQL, and integrated Razorpay.
  • AWS hosting with Route53 and Nginx configuration.
  • Django and MVT architecture.
  • Integrated Razorpay for secure online payments and refunds.
